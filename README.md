## Rask start

### Systemkrav
- PHP 8.1+
- MySQL 8.0.44-cll-lve
- XAMPP (anbefalt for lokal utvikling)

### Oppsett
Se [doc/SETUP_GUIDE.md](doc/SETUP_GUIDE.md) for fullstendig installasjonsguide.

---

## Stack

- **Backend:** PHP 8.1+ (strict_types)
- **Database:** MySQL 8.0.44-cll-lve (PDO)
- **Frontend:** HTML5, minimal CSS, vanilla JS
- **Auth:** Session-basert med "Remember me"
